IN THIS FOLDER...

2D-TrainLU4IterVariableNoise.py -> A python code to train a 4 iterations Learned Update algorithm starting from a 3 iteration one according to the progressive learning strategy

LearnedUpdate3Iter.tar -> A .tar file with the parameters and the optimizer state of a 3 iterations Learned Update algorithm 

LearnedUpdate4Iter.tar -> A .tar file with the parameters and the optimizer state of a 4 iterations Learned Update algorithm 

IQphantom.sino.mnc -> A file containing direct sinograms obtained from a miniPET measure

2D-LU4IterResultsEvaluationVariableNoise.ipynb -> A jupyter notebook to evaluate a 1, 2, 3, 4 iterations Learned Update algorithm results
