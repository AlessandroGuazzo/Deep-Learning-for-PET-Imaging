IN THIS FOLDER...

2D-TrainLPD3IterVariableNoise.py -> A python code to train a 3 iterations Learned Primal-Dual algorithm starting from a 2 iteration one according to the progressive learning strategy

LearnedPrimalDual2Iter.tar -> A .tar file with the parameters and the optimizer state of a 2 iterations Learned Primal-Dual algorithm 

LearnedPrimalDual3Iter.tar -> A .tar file with the parameters and the optimizer state of a 3 iterations Learned Primal-Dual algorithm 

IQphantom.sino.mnc -> A file containing direct sinograms obtained from a miniPET measure

2D-LPD3IterResultsEvaluationVariableNoise.ipynb -> A jupyter notebook to evaluate a 3 iterations Learned Primal-Dual algorithm results
