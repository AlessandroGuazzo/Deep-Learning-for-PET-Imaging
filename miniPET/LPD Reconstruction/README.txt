IN THIS FOLDER...

2D-Train3LPDminiPETHybrid.py -> a .py file to train the LPD algorithm with the Hybrid approach

2D-Train3LPDminiPETOnly.py -> a .py file to train the LPD algorithm with the miniPET only approach

2D-TestLPD3miniPET.ipynb -> a Jupyter notebook to test the LPD performances on miniPET data

3LPDHybrid.tar -> a.tar file with the parameters of the best performing model trained with the Hybrid approach

3LPDOnly.tar -> a .tar file with the parameters of the best performing model trained with the miniPET only approach
