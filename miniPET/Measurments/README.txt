IN THIS FOLDER...

MeasureSimulatorP1.xlsx -> An excel file used to set up measurments with the first phantom. It is possible to predict single objects activity concentrations and the total phantom activity

MeasureSimulatorP2.xlsx -> An excel file used to set up measurments with the second phantom. It is possible to predict single objects activity concentrations and the total phantom activity

MeasureSimulatorP3.xlsx -> An excel file used to set up measurments with the third phantom. It is possible to predict single objects activity concentrations and the total phantom activity

Measurements Description.txt -> A file used to report the effective activity levels of all performed measurments

processing.sh -> A linux bash file to process the .clm.lr5 file given as output from the miniPET acqusition
